# IoT Vending Monitoring System

This project simulates an IoT monitoring system for vending machines.
It focuses on stock level monitoring and low-stock alerts.

## What it includes
- Sensor data simulation
- Stock level monitoring
- Alert logic for low inventory

## Why this project
This project connects embedded systems thinking with software logic.
It reflects my experience with Arduino, sensors, and real-time systems.

## Tools used
- Python
- Basic IoT simulation logic
